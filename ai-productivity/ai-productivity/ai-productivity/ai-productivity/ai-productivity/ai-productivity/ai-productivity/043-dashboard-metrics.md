# Dashboard Metrics

## Purpose

AI can suggest useful metrics that support decisions instead of vanity tracking.

This note is part of an AI productivity knowledge base. The goal is to use AI as a practical assistant for planning, writing, reviewing, learning, and decision support.

## Practical Use

Use this workflow when a task feels unclear, repetitive, too large, or difficult to organize. AI is most useful when the user provides clear context and asks for a structured output.

## Suggested Workflow

1. Define the real goal.
2. Add the important context and constraints.
3. Ask for a clear output format.
4. Review the result instead of copying blindly.
5. Edit the output for accuracy, tone, and usefulness.
6. Save reusable parts as a template or checklist.

## Example Prompt

```text
Help me apply dashboard metrics to my current work. Give me a checklist, risks, and next actions.
```

## Quality Checklist

- Is the goal clear?
- Is the output practical?
- Are assumptions visible?
- Are risks or limitations mentioned?
- Is there a next action?
- Does the final version still need human review?

## Final Note

AI productivity is not about producing more noise. It is about turning information into clearer decisions, better systems, and useful action.
